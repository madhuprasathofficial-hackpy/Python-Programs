# 🌐 Flask Framework Theory

## What Is Flask?

Flask is a lightweight Python web framework used to create websites and web applications.

### Flask Features

- Simple and Fast
- Easy for Beginners
- Flexible Framework
- Used for APIs and Websites

---

# ⚙️ Flask Step By Step Process

## Step 1️⃣ Install Flask

```bash
pip install flask
```

---

## Step 2️⃣ Create Flask File

Create a file named:

```bash
app.py
```

---

## Step 3️⃣ Write Flask Program

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Welcome To MadhuPrasath Flask Project"

if __name__ == '__main__':
    app.run(debug=True)
```

---

## Step 4️⃣ Run Flask Application

```bash
python app.py
```

---

## Step 5️⃣ Open Browser

```bash
http://127.0.0.1:5000
```

# 📧 Automatic Email Sender Theory

## What Is SMTP?

SMTP means:

```bash
Simple Mail Transfer Protocol
```

It is used to send emails automatically using Python.

---

# ⚙️ Email Sending Process

## Step 1️⃣ Import SMTP Library

```python
import smtplib
```

---

## Step 2️⃣ Login Gmail Account

Use:

- Gmail Address
- App Password

---

## Step 3️⃣ Send Email

```python
import smtplib

server = smtplib.SMTP('smtp.gmail.com', 587)

server.starttls()

server.login("yourmail@gmail.com", "app_password")

server.sendmail(
    "yourmail@gmail.com",
    "receiver@gmail.com",
    "Hello From MadhuPrasath Python Project"
)

print("Email Sent Successfully")
```
